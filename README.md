## 파일 송수신 프로그램


- ReceiverThread: server에서 파일을 받아오는 스레드

- SenderThread: client에서 파일을 보내는 스레드

- ClientUI/ServerUI: Sender, Receiver에서 사용 가능한 GUI

- ReceiverMain/SenderMain: main문. ReceiverMain를 먼저 실행해야 동작
